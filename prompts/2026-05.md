ANNUAL ROTATION SYSTEM
════════════════════════════

Rotate layouts so each issue feels distinct but the magazine is recognizable.
Assign variant by issue number: odd=A or C, even=B or D, alternate each quarter.

VARIANT A — Left-heavy, vertical rhythm broken by one full-bleed horizontal band
  · Headlines start x=50, body col starts x=50 (wide) + x=420 (narrow)
  · Ghost letter top-right corner
  · Stat boxes: horizontal strip across full width

VARIANT B — Right-anchored titles, asymmetric image left
  · Headlines start x=220, text-anchor=end at x=744
  · Large image placeholder left side, text floats right
  · Ghost number bleeds left edge (x=-40)
  · Stat boxes: vertical stack left margin

VARIANT C — Diagonal tension, text overlaps image zone
  · One diagonal SVG rule across hero (not horizontal)
  · Headlines x=50, but rotate one word -2deg (CSS transform)
  · Pull-quote placed in image zone overlap (z-index above placeholder)
  · Stat boxes: 3-column grid, centered, narrow

VARIANT D — Sparse, maximum white space, single column
  · Only one body column (width 460px, x=50)
  · Headlines enormously tall (100–130px)
  · Zero stat boxes — replace with one large pull-quote
  · Image placeholder: portrait format only, right side

════════════════════════════
SVG DECORATION SYSTEM
════════════════════════════

Every page needs ≥ 3 custom SVG elements. 
ZERO emoji. ZERO icon fonts. ZERO Unicode decorative chars.
Build everything from: line, circle, ellipse, rect, path, text.

Required elements per page (rotate which ones):
□ Ghost letterform: single letter, stroke only, 300–500px, opacity 0.04–0.07
□ Registration crosshair: 2 lines + outer circle + 4px center dot
□ Orbital ring: ellipse, fill none, stroke 0.5px, opacity 0.15–0.2
□ Arc fragment: path A command, fill none, stroke 0.6px gold
□ Measurement ticks: series of 5–7 small tick marks along a line
□ Crop marks: L-shaped pairs at 2 corners (not all 4 — asymmetric)
□ Fine grid overlay: repeating-linear-gradient, 60px cells, opacity 0.05
□ Dotted connector: path, stroke-dasharray="2 4", from image placeholder to caption

Decorative lines:
  · All rules: stroke-width 0.3–0.5px
  · Double rules: two lines 3px apart, second at 0.5× opacity
  · Diagonal rules: not horizontal — slight angle (y varies 10–20px across width)
  · Never draw boxes around content sections — use single-sided borders only

Stat boxes: stroke 0.5px, opacity 0.25, border-radius 0 (flat, architectural)
Pull-quote: left border only, 0.8px gold, border-radius 0
